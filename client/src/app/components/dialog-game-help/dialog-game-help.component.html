Placer une lettre sur le plateau de jeu
arrow_right_alt arrow_right_alt

Pour placer une lettre, il faudra CLIQUER sur le plateau de jeu et ÉCRIRE les lettres à placer avec le clavier. Pour finaliser le placement, pressez le boutton ENTER.

Note : Il faut attendre son tour avant de placer des lettres
Placer une lettre dans une direction
arrow_right_alt

Par défaut, le PREMIER CLIQUE sur une case permet un placement HORIZONTAL.

arrow_right_alt

Le SECOND CLIQUE sur une case permet un placement VERTICAL.
Enlever des lettres

Pour enlever une lettre non finalisée, utiliser la touche BACKSPACE.

arrow_downward

Pour enlever toutes les lettres non finalisées, utiliser la touche ESCAPE.

arrow_downward
Déplacer une lettre dans le chevalet

Pour selectionner une lettre à déplacer, faites un CLIQUE GAUCHE dessus.

Pour déplacer la lettre, utiliser les touches ← pour déplacer à gauche et → pour déplacer à droite.

Il est aussi possible d'utiliser le SCROLL de la souris pour deplacer la lettre

arrow_downward
Échanger des lettres du chevalet

Pour selectionner des lettres à échanger, faites des CLIQUES DROIT sur eux.

Pour finaliser l'échange, cliquer sur ECHANGER

arrow_downward
Utilisation des commandes

Il est possible d'utiliser la boite de dialogue pour effectuer des commandes.

Pour trouver tout les commande, il suffit d'écrire !aide dans la boite de dialogue.
