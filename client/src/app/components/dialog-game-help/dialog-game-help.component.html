Placer une lettre sur le plateau de jeu
arrow_right_alt arrow_right_alt

Pour placer une lettre, il faudra CLIQUER sur le plateau de jeu et ÉCRIRE les lettres à placer avec le clavier. Pour finaliser le placement, pressez le boutton ENTER.

Note : Il faut attendre son tour avant de placer des lettres
Placer une lettre dans une direction
arrow_right_alt

Par défaut, le PREMIER CLIQUE sur une case permet un placement HORIZONTALE.

arrow_right_alt

Le SECOND CLIQUE sur une case permet un placement VERTICAL.
Enlever des lettres

Pour enlever une lettre non finalisée, utiliser la touche BACKSPACE.

arrow_downward

Pour enlever tout les lettres non finalisées, utiliser la touche ESCAPE.

arrow_downward
Utilisation des commandes

Il est possible d'utiliser la boite de dialogue pour effectuer des commandes.

Pour trouver tout les commande, il suffit d'écrire !aide dans la boite de dialogue.
